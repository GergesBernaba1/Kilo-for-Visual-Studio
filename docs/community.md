# Community and Support

## Join the Community

- Discord: `https://discord.gg/your-kilo-community` (placeholder link, replace with actual invite).
- Forum: `https://community.kilo.ai` (or use GitHub Discussions in this repo).

## Support Channels

- Issues: Use GitHub Issues for bugs and feature requests.
- Discussions: Use GitHub Discussions for usage questions, how-to guides, and plugin suggestions.
- Slack/Teams: Optional third-party channel (if active).

## Reporting Bugs

Please include:
- VS version (e.g., `Visual Studio 2022 17.8`)
- Operating system (Windows version)
- Extension version (from VS extension manager)
- Steps to reproduce
- Logs from `%LOCALAPPDATA%\Kilo.VisualStudio\Logs`

## Feature Requests

- Describe the desired behavior.
- Provide a short use case.
- Mention any related upstream Kilo capabilities you want to mirror.

## Contributor Roles

- Maintainers: approve and merge PRs
- Contributors: code changes, documentation updates, test coverage
- Reviewers: sanity checks and feature validation on VS2019/VS2022
